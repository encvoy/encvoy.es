## ğŸ”‘ Authentication and Login methods

encvoy ID supports a wide range of authentication methods,
including passwordless and phishing-resistant options.

Built on widely adopted standards:

- OpenID Connect (OIDC)  
- OAuth 2.0  

### Standard methods
- Username and password  
- Email-based login  

### Social and external providers
- Google, GitHub and other providers  
- External identity providers  

### Strong and passwordless methods
- WebAuthn / Passkeys  
- mTLS (client certificates)  
- TOTP / HOTP one-time passwords  

Supports multi-factor authentication (MFA) combining multiple verification factors.

Helps meet higher security and compliance requirements.
Encvoy ID is designed to support a gradual transition toward passwordless authentication.
